# リードミー
読み終わった**IT技術書のレビューを投稿**し、情報を共有してユーザー同士で**IT技術の勉強教材の選別に役立てる**ことを目的としたアプリです。  

![home](https://github.com/user-attachments/assets/ab0c7039-94b4-40bf-8bb5-fa69ceb59827)
